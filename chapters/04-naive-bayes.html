4.4 Key Findings

How Gaussian Naïve Bayes Works

Gaussian Naïve Bayes is a probabilistic classifier that combines two ideas:

  1. Bayes’ theorem: P(class | features) ∝ P(features | class) × P(class)
  2. Naïve independence assumption: treat each feature as conditionally independent given the class, so the joint likelihood factors into a product:

\[P(x_1, x_2, x_3, x_4 \mid class) = \prod_{i=1}^{4} P(x_i \mid class)\]

Each conditional likelihood P(xᵢ | class) is modeled as a Gaussian distribution whose mean μ and standard deviation σ are estimated directly from the training data for each class.

  • Training: compute μ and σ for every (class, feature) pair → store as a lookup table.
  • Prediction: for a new flower, evaluate the Gaussian PDF at each observed feature value, multiply across features, scale by the class prior, and pick the class with the highest score. Log-probabilities are used in practice to avoid numerical underflow from multiplying many small numbers.

#| include: false
import random
import numpy as np
import pandas as pd
import statistics
import plotly.graph_objects as go
import plotly.express as px
from sklearn.model_selection import train_test_split
from sklearn.metrics import confusion_matrix

# ── Load Iris (UCI with sklearn fallback) ─────────────────────────────────────
try:
    from ucimlrepo import fetch_ucirepo
    iris = fetch_ucirepo(id=53)
    X = iris.data.features
    y = iris.data.targets
    X.columns = ['sepal length', 'sepal width', 'petal length', 'petal width']
except Exception:
    from sklearn.datasets import load_iris
    _iris = load_iris(as_frame=True)
    X = _iris.data
    X.columns = ['sepal length', 'sepal width', 'petal length', 'petal width']
    _mapping = {0: 'Iris-setosa', 1: 'Iris-versicolor', 2: 'Iris-virginica'}
    y = _iris.target.map(_mapping).rename('class').to_frame()

# ── Train / test split (same seed as original notebook) ──────────────────────
random.seed(0)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.20, random_state=0)
Xy_train = pd.concat([X_train, y_train], axis=1)

classes  = list(pd.unique(y_train['class']))
features = list(X_train.columns)

# ── Class-wise mean & sd ──────────────────────────────────────────────────────
mean_dict, sd_dict, prior_dict = {}, {}, {}
for c in classes:
    X_c = X_train[y_train['class'] == c]
    prior_dict[c] = len(X_c) / len(X_train)
    mean_dict[c]  = {f: statistics.mean(X_c[f]) for f in features}
    sd_dict[c]    = {f: statistics.stdev(X_c[f]) for f in features}

# ── Gaussian PDF ──────────────────────────────────────────────────────────────
def normal_dist(x, mean, sd):
    return (1 / (sd * np.sqrt(2 * np.pi))) * np.exp(-0.5 * ((x - mean) / sd) ** 2)

# ── From-scratch GNB predictions ─────────────────────────────────────────────
y_pred = []
for i in range(len(X_test)):
    x = X_test.iloc[i]
    best_class, best_score = None, None
    for c in classes:
        score = np.log(prior_dict[c])
        for f in features:
            p = normal_dist(x[f], mean_dict[c][f], sd_dict[c][f])
            score += np.log(max(p, 1e-300))
        if best_score is None or score > best_score:
            best_score, best_class = score, c
    y_pred.append(best_class)

y_true_list = list(y_test['class'])
accuracy = round(sum(a == b for a, b in zip(y_true_list, y_pred)) / len(y_pred), 4)

# ── Manual calculation for the example point ─────────────────────────────────
ex = {'sepal length': 7.3, 'sepal width': 2.9, 'petal length': 6.3, 'petal width': 1.8}
prob_general = 0.333
manual_rows = []
for f, val in ex.items():
    row = {'Feature': f, 'Value': val}
    for c in classes:
        p = round(normal_dist(val, mean_dict[c][f], sd_dict[c][f]), 10)
        row[c.replace('Iris-', '')] = p
    manual_rows.append(row)
manual_df = pd.DataFrame(manual_rows)

joint = {}
for c in classes:
    score = np.log(prob_general)
    for f in features:
        p = normal_dist(ex[f], mean_dict[c][f], sd_dict[c][f])
        score += np.log(max(p, 1e-300))
    joint[c.replace('Iris-', '')] = round(np.exp(score), 8)

## How Gaussian Naïve Bayes Works

Gaussian Naïve Bayes is a probabilistic classifier that combines two ideas:

1. **Bayes' theorem:** P(class | features) ∝ P(features | class) × P(class)
2. **Naïve independence assumption:** treat each feature as conditionally independent given the class, so the joint likelihood factors into a product:

$$P(x_1, x_2, x_3, x_4 \mid class) = \prod_{i=1}^{4} P(x_i \mid class)$$

Each conditional likelihood P(xᵢ | class) is modeled as a **Gaussian distribution** whose mean μ and standard deviation σ are estimated directly from the training data for each class.

- **Training:** compute μ and σ for every (class, feature) pair → store as a lookup table.
- **Prediction:** for a new flower, evaluate the Gaussian PDF at each observed feature value, multiply across features, scale by the class prior, and pick the class with the highest score. Log-probabilities are used in practice to avoid numerical underflow from multiplying many small numbers.
